# ☕ Cafe Sales Data Cleaning

Exercício de limpeza e tratamento de dados utilizando Polars e Python.

O projeto foi desenvolvido com um dataset do Kaggle contendo dados de vendas com inconsistências propositalmente inseridas para prática de Data Cleaning e ETL.

## 🔧 Tecnologias

* Python
* Polars
* Jupyter Notebook

## 📥 Dataset

[Kaggle - Cafe Sales Dirty Data for Cleaning Training](https://www.kaggle.com/datasets/ahmedmohamed2003/cafe-sales-dirty-data-for-cleaning-training)

## 📌 Tratamentos realizados

* Remoção de valores nulos
* Tratamento de dados inválidos
* Conversão de tipos
* Limpeza de datas
* Padronização de colunas
* Exportação do dataset limpo

## 🚀 Objetivo

Praticar conceitos de:

* ETL
* Data Cleaning
* Manipulação de dados
* Processamento com Polars
